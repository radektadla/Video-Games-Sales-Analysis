# Video-Games-Sales-Analysis
Video games physical sales analysis using Excel (data cleaning, analysis & visualization).

## 📝 Short Description
This project focuses on exploratory data analysis of video games sales using Excel, with an emphasis on physical copies.

The goal of the analysis was to explore global and regional sales trends, dominant genres, and platform performance, providing answers to several analytical questions, including:
- Which games and genres sold best globally and regionally
- How have total sales evolved over time and potential periods of stagnation
- Sales distribution by console and genre
- Insights about regional differences in games popularities

All insights were derived from data cleaning, analysis, and visualization in Excel, with charts and tables highlighting key trends.

## 🛠️ Tools used
- Microsoft Excel - main tool used for data analysis and visualization through pivot tables, formulas and conditionally formated charts
- Power Query - data cleaning and transformation

## 💾 Data Source
- Source: Kaggle, via ASANICZKA
- File format: CSV
- Data structure: One tables
- Number of records: 64,016
- Date period: 1971–2024

## ❔ Key Questions
- Which games sold the best globally? Does this translate across all regions?
- Which game release year generated the highest total global sales?
- Is the video game market growing over time or are there periods of stagnation/ decline?
- Which genres dominate on specific consoles?

## 📊 Results/ Insights
**1. Top Selling Games Globally and Regionally**

**Top Selling Games Globally**
![Top Selling Games Globally](https://github.com/radektadla/Video-Games-Sales-Analysis/blob/main/Result%20Preview/Top%20Selling%20Games%20Globally.png)

Short summary: GTA V leads global sales with more than twice the units of the second-best game.
The Call of Duty series is also poular, with 7 titles in the top 10.

**Top Selling Games Regionally**
![Top Selling Games Regionally](https://github.com/radektadla/Video-Games-Sales-Analysis/blob/main/Result%20Preview/Top%20Selling%20Games%20Per%20Region.png)

Short summary: GTA V leads global sales with an exception in Japan, where Hot Shots Golf tops the charts. Regional favorites differ: Call of Duty dominates in North America, while FIFA tends to be more popular in Europe.

**2. Sales Over Time**

![Sales Over Time](https://github.com/radektadla/Video-Games-Sales-Analysis/blob/main/Result%20Preview/Sales%20Over%20Time.png)

Short summary: Games released in 2008 recorded the highest total global sales, reaching 538,11 million sold copies, while games released between 2007 and 2011 achieved the strongest sales performance. There is a significant increase in sales for games released after 2000, and a decline in sales for games released after 2014.

**3. Dominant Genre by Console**

![Dominant Genre by Console](https://github.com/radektadla/Video-Games-Sales-Analysis/blob/main/Result%20Preview/Dominant%20Genre%20by%20Console.png)

Short summary: Action games dominate PlayStation and Xbox consoles, making it the most popular genre overall.
PCs are led by simulation games, while the Wii platform is dominated by music games.
